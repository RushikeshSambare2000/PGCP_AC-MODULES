Serialization
-----------------

- To store/save/transfer object onto the network/secondary storage.

FileInputStream
ObjectInputStream -- readObject();

FileOutputStream
ObjectOutputStream -- writeObject();

Object which is to be serialized must implement the marker interface Serializable.

class Student implements Serializable
{
}

So, now Student objects can be serialized.

Marker Interface: Serializable, Cloneable, Remote
Marker Interface: Interface without any abstract method.

Ex:
interface MyInterface
{
}
